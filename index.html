UK Energy Trends: Real-Time Data Engineering & Fuel Mix Analysis

This project explores the UK's electricity generation trends using real-time data from the Carbon Intensity API. It features a complete data pipeline built with Python, Apache Airflow, and SQLite to collect, store, and backfill hourly energy mix data. The project automates ingestion, enables scalable analysis, and uncovers insights on fuel usage patterns by hour, day, season, and fuel type. Through clean visualisations and structured analytics in Jupyter Notebooks, it highlights how the UK balances clean versus fossil energy sources

Tech: Python, Apache Airflow, SQLite, Pandas, Matplotlib, Jupyter Notebook, Git, GitHub, Carbon Intensity API

Image Similarity Based Recommendation System

This project implements a content-based recommendation system that suggests visually similar movies using poster image embeddings. Leveraging the TMDB API, the pipeline extracts movie posters and applies a pre-trained ResNet50 model for deep feature extraction. Autoencoders are then used to reduce dimensionality, optimising for performance without sacrificing accuracy. Cosine similarity metrics drive the recommendation engine, clustering films with related visual aesthetics. The system enhances user experience by enabling discovery of movies based on visual style rather than just metadata or user ratings.

Tech: Python, TensorFlow/Keras, ResNet50, Autoencoders, Cosine Similarity, TMDB API, Jupyter Notebook, Git, GitHub
